API - Dokumentation

Allgemeines

Diese Applikation fängt URL-Aufrufe auf einem HTTP-Port ab und leitet davon abhängig, passende Datenbank Statements weiter an eine MariaDb.

Die Basisurl der API

https://rudolf-histel.de/api

Datenabhängige URL-Zusätze

Je nach gewünschten Daten müssen an die Basisurl passende Zusätze angehangen werden:

  • Produkte: https://rudolf-histel.de/api/products

Danach folgende weitere Zusätze je nach dem was Sie mit den Daten konkret machen möchten. Das wird Ihnen in den folgenden Abschnitten unten stehend genau erklärt.

Produkte

Datentypen

Alle Produkte werden unter dem Domain zusatz
https://rudolf-histel.de/api/products gefunden.

Schöne Beispiel-Darstellung der Datentypen:
[
	{
		"id": number,
		"name": string,
		"price": number
	}
]

Alle Produkte auslesen:

https://rudolf-histel.de/api/products
